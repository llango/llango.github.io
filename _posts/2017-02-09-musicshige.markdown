窗外,<br/>
暖暖地,<br/>
拂动笑容的夕阳。<br/>
我的脸上,<br/>
仿佛间,<br/>
感受到蠢蠢萌动的春天在欢呼.<br/>
<br/>
初见,<br/>
每一颗在我周遭跳动的心:<br/>
温馨---忆恋的亲吻,<br/>
像阵春风触动着我冬天的短梦,<br/>
梦里是个孤独的少年坐在灯塔上凝注星空.<br/>
<br/>
相视,<br/>
每一次插肩而过,<br/>
不经意地,<br/>
走进了眼睛,走进了内心.<br/>
这偶然,<br/>
成了记忆里一个个温柔的倩影.<br/>
<br/>
暖暖地<br/>
窗外<br/>
<br/>
----凉葱落
